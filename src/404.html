Global 404

Note: To go to any 404 we can add "#404-404Number" in the url. For example, to go to 404-01 add "#404-01", the url will be "https://globalcomponents.netlify.app/404.html#404-01"

Note variables: For 404 modifiers we can edit any variable. We can change any variable at any breakpoint using the mixin modifier except variables that depend on an existing class (btns, fonts, pills...). If we want to change a font in a modifier, the default font must be the first one (starting with a).

Note button/links/pills: In all places where there is a link, pill or a button it can be any of them
