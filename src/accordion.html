Global Accordion

Note: To go to any accordion we can add "#accordion-accordionNumber" in the url. For example, to go to accordion-02 add "#accordion-02", the url will be "https://globalcomponents.netlify.app/accordion.html#accordion-02"

Note variables: For accordion modifiers we can edit any variable. We can change any variable at any breakpoint using the mixin modifier.

Note border: For accordion we just set the border-bottom.
