Global Buttons

Note: To go to any button we can add "#btn-btnNumber" in the url. For example, to go to btn-02 add "#btn-02", the url will be "https://globalcomponents.netlify.app/buttons.html#btn-02"

Note variables: For button modifiers we can edit any variable. We can change any variable at any breakpoint using the mixin modifier.

Note border: We can set different border-color, border-style or border-width for each side (or add border just to one side).
