Global Cards

Note: To go to any card we can add "#card-cardNumber" in the url. For example, to go to card-32 add "#card-32", the url will be "https://globalcomponents.netlify.app/cards.html#card-32"

Note variables: All the variables in black are necessary to build the card, the variables in purple are optional. For card modifiers we can edit any variable (in black or purple). We can change any variable at any breakpoint using the mixin modifier.

Note hover options: change background-color, change title-color, add box-shadow, transform card (translate, zoom...), image zoom

Note button/links: In all places where there is a link or a button it could be either
