Global Cards

Note: To go to any card we can add "#card-cardNumber" in the url. For example, to go to card-32 add "#card-32", the url will be "https://globalcomponents.netlify.app/cards.html#card-32"

Note variables: For card modifiers we can edit any variable. We can change any variable at any breakpoint using the mixin modifier.

Note border: For cards with border we can set different border-color, border-style or border-width for each side (or add border just to one side).

Note hover options: change background-color, change title-color, add box-shadow, transform card (translate, zoom...), image zoom. *If the title is a c--content we can't change the color on hover.

Note button/links: In all places where there is a link or a button it could be either
