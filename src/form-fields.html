Global Form Fields

Note: To go to any form field we can add "#form-formItem-formNumber" in the url. For example, to go to form-input-01 add "#form-input-01", the url will be "https://globalcomponents.netlify.app/form-fields.html#form-input-01"

Note variables: For form fields modifiers we can edit any variable. We can change any variable at any breakpoint using the mixin modifier except variables that depend on an existing class (fonts...) and $font-size-mobile variable. If we want to change a font in a modifier, the default font must be the first one (starting with a).

Note border: For form fields with border we can set different border-color, border-style or border-width for each side (or add border just to one side).
