# BTN

## btn-1

Html for a button

```sh
<button class="g--btn-1">button 1</button>
```

Html for a link

```sh
<a href="#" class="g--btn-1">button 1</a>
```

### editable variables
- $font
- $border-width
- $border-color
- $text-color
- $border-color-hover
- $text-color-hover
