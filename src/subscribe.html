Global subscribe

Note: To go to any subscribe form we can add "#subscribe-fsubscribeNumber" in the url. For example, to go to subscribe-01 add "#subscribe-01", the url will be "https://globalcomponents.netlify.app/subscribe.html#subscribe-01"

Note variables: For subscribe modifiers we can edit any variable. We can change any variable at any breakpoint using the mixin modifier except variables that depend on an existing class (fonts...) and $font-size-mobile variable. If we want to change a font in a modifier, the default font must be the first one (starting with a).

Note border: For subscribe with border we can set different border-color, border-style or border-width for each side (or add border just to one side).
